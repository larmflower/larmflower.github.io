# My Blog
# built with the Particle Jekyll Theme

![](./particle.jpg)

This is a simple and minimalist blog built with a Jekyll template designed for developers that want to show of their portfolio.

The Site features:

- Gulp
- SASS
- Sweet Scroll
- Particle.js
- BrowserSync
- Font Awesome and Devicon icons
- Google Analytics
- Info Customization
